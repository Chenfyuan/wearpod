# Google Play Release Checklist

## Required before upload
- [ ] Finalize formal release keystore backup
- [ ] Increment `versionCode`
- [ ] Confirm `versionName`
- [ ] Build signed `bundleRelease`
- [ ] Verify privacy policy URL is reachable
- [ ] Verify support email is monitored

## Store listing
- [ ] Paste English store listing copy
- [ ] Paste Simplified Chinese store listing copy
- [ ] Upload Wear OS screenshots
- [ ] Upload feature graphic
- [ ] Upload 512x512 high-res icon
- [ ] Confirm category is `Music & Audio`

## Policy / compliance
- [ ] Complete Data safety form
- [ ] Complete Content rating questionnaire
- [ ] Confirm no misleading screenshots
- [ ] Confirm standalone Wear OS behavior matches listing claims

## Product QA before rollout
- [ ] First launch works
- [ ] Language switching works
- [ ] Phone import works
- [ ] Phone export works
- [ ] Offline download works
- [ ] Offline playback works
- [ ] Bluetooth / speaker output switching works
- [ ] App handles no-network cases clearly

## Post-upload
- [ ] Fill release notes
- [ ] Start internal or closed testing
- [ ] Review pre-launch report
- [ ] Fix stability, accessibility, or policy issues before production rollout
