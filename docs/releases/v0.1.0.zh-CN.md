# WearPod v0.1.0

[English release notes](./v0.1.0.md)

这是 SJTech 发布的 WearPod 首个公开原型版本。

## 版本亮点

- 面向圆形手表交互优化的独立 Wear OS 播客应用
- 支持直接在手表上导入公开 RSS
- 基于 `Media3` 的 watch-first 播放链路
- 支持下载队列、失败重试和离线播放
- 首页基于收藏内容进行聚合
- 支持后台订阅刷新和自动下载设置
- 播放器内置睡眠定时

## 本版本包含的能力

### 核心页面

- 首页
- 订阅
- 导入订阅
- 播客详情
- 播放器
- 离线
- 下载与刷新设置

### 播放能力

- 继续播放
- 队列上一集 / 下一集
- 快退 / 快进
- 倍速切换
- `15 / 30 / 60` 分钟睡眠定时

### 离线能力

- 单集下载
- 播客整包下载
- 下载队列
- 下载失败重试
- 清空本地缓存

### 节目管理

- 标记已播 / 未播
- `全部`、`未播`、`已下载` 三种筛选
- 收藏播客并在首页展示

## 技术说明

- Kotlin + Compose for Wear OS
- Media3 负责播放
- WorkManager 负责下载与周期刷新
- 使用应用私有目录下的 JSON 文件做持久化

## 当前限制

- 仅支持公开 RSS
- 不支持私有或付费订阅鉴权
- 尚未支持 OPML 导入导出
- 没有云端同步
- 当前仍是文件持久化，后续可能迁移到 Room/DataStore

## 本地验证
