# WearPod v0.1.0

[中文版](./v0.1.0.zh-CN.md)

Initial public prototype release of WearPod by SJTech.

## Highlights

- Standalone Wear OS podcast app focused on round-watch interaction
- Direct public RSS import on the watch
- Watch-first playback flow with `Media3`
- Offline downloads with queue, failure state, and local playback
- Favorites-based home surface
- Background subscription refresh and auto-download controls
- Sleep timer inside the player

## Included in this release

### Core product flows

- Home
- Subscriptions
- Import RSS
- Podcast detail
- Player
- Downloads
- Download and refresh settings

### Playback

- Continue listening
- Previous / next item in queue
- Replay / forward seek
- Playback speed switching
- Sleep timer for `15 / 30 / 60` minutes

### Offline

- Per-episode download
- Download all episodes from a podcast
- Download queue
- Download failure retry
- Clear local cache

### Episode management

- Mark played / unplayed
- Filter by `All`, `Unplayed`, and `Downloaded`
- Favorite subscriptions to surface them on home

## Technical notes

- Kotlin + Compose for Wear OS
- Media3 for playback
- WorkManager for downloads and periodic refresh
- JSON file persistence in app-private storage

## Known limitations

- Public RSS only
- No private or paid feed authentication
- No OPML import/export
- No cloud sync
- Persistence is still file-based and may later move to Room/DataStore
